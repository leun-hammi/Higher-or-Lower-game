🔥 Higher or Lower Guessing Game

A fun command-line Python game where players guess which celebrity, athlete, brand, or public figure has more social media followers.

📖 About

In this game, two randomly selected profiles are shown.

The player must guess whether Person A or Person B has the higher follower count.

For every correct answer:

- The score increases by 1.
- The winning profile continues to the next round.

The game ends when the player makes an incorrect guess.

🚀 Features

- Random profile selection
- Score tracking
- Multiple rounds
- Input validation
- Clear screen effect between rounds
- Large dataset of celebrities, athletes, entrepreneurs, and organizations

🛠 Technologies Used

- Python
- Random Module

🎮 How to Play

1. Compare two profiles.
2. Type:
   - "A" for Profile A
   - "B" for Profile B
3. If your guess is correct:
   - Your score increases.
   - The game continues.
4. If your guess is wrong:
   - The game ends.
   - Your final score is displayed.

🏆 Scoring System

- Correct Guess = +1 Point
- Wrong Guess = Game Over

Try to achieve the highest score possible!
